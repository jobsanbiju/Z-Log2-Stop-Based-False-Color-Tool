# Z-Log2 Stop-Based False Color Tool

Exposure monitoring LUT for Z CAM Z-Log2 footage, styled around EL Zone stop-based false color.

![Z-Log2 Stop-Based False Color Guide](zlog2_stop_false_color_guide.png)

## Included

- `Z-Log2_to_El-Zone-like_jobsanbiju-v1.1.cube`
- `zlog2_stop_false_color_guide.png`

## Notes

- Designed for monitoring exposure, not final color grading.
- 18% middle grey is centered around approximately 40 IRE, with a practical grey zone around 37-43 IRE.
- The LUT is intended for Z CAM Z-Log2 material.
